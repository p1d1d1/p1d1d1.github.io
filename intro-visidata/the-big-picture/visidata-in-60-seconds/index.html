VisiData in 60 secondi¶

Ecco un’introduzione a VisiData in tre passi.

Passo 1: Usare vd per aprire un file di dati¶

Scarica faa-wildlife-strikes.csv, un set di dati delle collisioni di velivoli riportati alla Federal Aviation Administration tra il 2010 e la metà de 2016.

Da terminale portati nella cartella in cui hai scaricato il file, quindi lancia il seguente comando:

vd faa-wildlife-strikes.csv

Passo 3: Leggere la pagina del manuale di VisiData¶

La “guida rapida di riferimento di VisiData” elenca tutti i comandi e le funzionalità di VisiData. Puoi leggerla online o accedervi da VisiData premendo il taso F1 o digitando Control-h:

vd(1)                        Quick Reference Guide                       vd(1)                      
NAME                     
     VisiData -- a terminal utility for exploring and arranging tabular data                        
SYNOPSIS                 
     vd [options] [input ...]                     
     vd [options] --play cmdlog [-w waitsecs] [--batch] [-o output] [field=value ...]               
DESCRIPTION              
     VisiData is a multipurpose tool built on the vdtui platform that can be used to                
     explore, clean, edit, and restructure data. Rows can be selected, filtered, and                
     grouped; columns can be rearranged, transformed, and derived via regex or Python               
     expressions; and workflows can be saved, documented, and replayed.                             
   REPLAY MODE           
     -p, --play=cmdlog       replay a saved cmdlog within the interface                             
     -w, --replay-wait=seconds                    
                             wait seconds between commands                                          
     -b, --batch             replay in batch mode (with no interface)                               
     -o, --output=file       save final visible sheet to file as .tsv                               
     --replay-movement       toggle --play to move cursor cell-by-cell                              
:

Note

Se apri il manuale dall’interno di VisiData, esso verrà lanciato nel programma “pager” del terminale — normalmente il programma less. Per spostarti all’interno di “less”:

Combinazione(i) di tasti Azione
Space / b Scroll avanti/indietro
/ + termine + Enter Cerca termine
n / N Va a occorrenza prossima/precedente di termine
q Esce e torna a VisiData

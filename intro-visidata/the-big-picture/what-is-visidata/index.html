Cos’è VisiData?¶

VisiData è uno strumento open source che consente di aprire, esplorare, riepilogare ed analizzare velocemente dati con il terminale.

Note

Non è necessario essere un programmatore per usare VisiData. Conoscere un po’ di Python, tuttavia, renderà l’uso di VisiData ancora più efficace.

Perché usare VisiData?¶

  • È veloce:
    • VisiData si apre in un batter d’occhio e può caricare svariati megabyte di dati quasi istantaneamente.
  • È agile:
    • VisiData rende facile la ricerca, il filtraggio, l’ordinamento ed il join di qualsiasi tabella.
  • È ottimo per avere velocemente un’idea di un nuovo set di dati:
    • Con pochi tasti è possibile analizzare qualsiasi colonna.
  • Funziona con svariati formati di dati:
    • CSV, Excel, JSON e altri.
  • Non è distruttivo:
    • VisiData non modificherà i tuoi dati grezzi a meno che non glielo dici esplicitamente.
  • È guidato da tastiera:
    • Non è necessario usare il mouse.

Quando non usare VisiData?¶

VisiData non è particolarmente adatto per i compiti seguenti:

  • Analisi complessa.
    • Probabilmente è meglio usare Python, R, o un altro linguaggio di programmazione.
  • Analisi “Literate”.
    • Anche se VisiData consente di salvare e riprodurre le analisi, non è così leggibile/flessibile come i notebooks Jupyter, R Markdown, o altri strumenti.
  • Analisi mission-critical.
    • VisiData è ben progettato, ma è ancora un software relativamente giovane.

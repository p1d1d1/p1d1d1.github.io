Un’introduzione a VisiData¶

VisiData è uno strumento open source che consente di aprire, esplorare, riassumere ed analizzare velocemente dati con il terminale. VisiData funziona con file CSV, fogli di calcolo Excel, database SQL e moltissime altre fonti. L’interfaccia ha l’aspetto seguente:

=============
Installazione
=============

Installare VisiData è alquanto semplice. Avrai bisogno di:

- Un computer con Linux o MacOS
- `Python 3 <https://www.python.org/downloads/>`_
- Un po' di dimistichezza con la linea di comando

Apri il terminale e lancia il comando:

::

    pip3 install visidata


VisiData è il nome del programma, ``vd`` è il comando da terminale corrispondente. Per verificare che l'installazione sia andata a buon fine, lancia:

::

    vd --version

Dovresti vedere qualcosa di simile a quanto segue:

    :file: ../raw_html/version.output.html

.. note::

   Per creare una "sandbox" di VisiData e delle sue dipendenze, puoi `installare il programma tramite pipx <https://gist.github.com/jsvine/4fd295176b4f5257dfb89b3898a21a7e>`_.
